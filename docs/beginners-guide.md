# 🚀 초보자 가이드 (Beginner's Guide)

> 프로그래밍을 모르셔도 괜찮습니다. 이 가이드만 따라하면 됩니다.

---

## SalmAlm이 뭔가요?

**나만의 AI 비서**입니다. ChatGPT처럼 대화할 수 있지만, 내 컴퓨터에서 돌아갑니다.

- 💬 자연어로 대화 (한국어 OK)
- 📁 내 파일 읽기/쓰기
- 🌐 웹 검색
- 🧠 여러 AI 모델 자동 선택 (비싼 모델은 어려운 질문에만)
- 💰 ChatGPT Plus($20/월)보다 **80% 저렴**

---

## 설치하기

### 필요한 것
- Python 3.10 이상 ([다운로드](https://python.org/downloads))
- AI API 키 하나 (아래 참고)

### API 키 받기 (무료 체험 가능)
| 제공사 | 가입 | 비고 |
|---|---|---|
| [Google AI Studio](https://aistudio.google.com/apikey) | 구글 계정으로 로그인 | **무료 티어 있음** — 처음 시작하기 좋음 |
| [Anthropic](https://console.anthropic.com/) | 회원가입 | $5 무료 크레딧 (신규) |
| [OpenAI](https://platform.openai.com/api-keys) | 회원가입 | 선불 충전 필요 |

### 설치 (터미널에 복사+붙여넣기)
```bash
pip install salmalm
```

### 실행
```bash
salmalm --open
```
브라우저가 자동으로 열립니다. 끝!

## 처음 해볼 것들

설치하고 나서 뭘 해야 할지 모르겠다면, 이것부터 해보세요:

### 1. 그냥 대화하기
```
안녕! 넌 뭘 할 수 있어?
```
AI가 자기가 할 수 있는 것들을 알려줍니다.

### 2. 웹 검색
```
오늘 서울 날씨 알려줘
```
자동으로 웹을 검색해서 답해줍니다.

### 3. 파일 다루기
```
바탕화면에 todo.txt 만들어줘. 내용은 "장보기, 운동하기"
```

### 4. 코드 도움
```
파이썬으로 가위바위보 게임 만들어줘
```

### 5. 명령어 보기
```
/help
```
카테고리별로 정리된 명령어 목록을 볼 수 있습니다.

## 자주 묻는 질문

### Q: 내 데이터는 안전한가요?
**네.** 모든 데이터는 내 컴퓨터(`~/SalmAlm` 폴더)에만 저장됩니다. 외부 서버로 전송되지 않습니다. API 키는 암호화되어 보관됩니다.

### Q: 비용이 얼마나 드나요?
API 사용량에 따라 다르지만, 일반적인 사용(하루 50회 대화)은 **$0.5~1.5/일** 정도입니다. 자동 라우팅이 간단한 질문은 저렴한 모델로 보내줍니다.

### Q: ChatGPT랑 뭐가 다른가요?
- ChatGPT: 월 $20 고정, OpenAI 모델만
- SalmAlm: 쓴 만큼만 지불, Anthropic/OpenAI/Google/로컬 모델 전부 사용 가능, 내 컴퓨터에서 동작

### Q: 에러가 나요!
```bash
salmalm doctor
```
자가진단을 실행하면 대부분의 문제를 찾아줍니다.

### Q: 업데이트는 어떻게?
```bash
salmalm --update
```
