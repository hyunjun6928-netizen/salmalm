# Features Module API
# 기능 모듈 API

The `salmalm.features` package contains all feature modules.

`salmalm.features` 패키지는 모든 기능 모듈을 포함합니다.

## Command & Control / 명령어 및 제어

| Module / 모듈 | Description / 설명 |
|---|---|
| `features.commands` | 30+ slash command handler / 슬래시 명령어 핸들러 |
| `features.abort` | Execution abort handler / 실행 중단 핸들러 |
| `features.fork` | Conversation branching / 대화 분기 |

## AI & Intelligence / AI 및 지능

| Module / 모듈 | Description / 설명 |
|---|---|
| `features.rag` | RAG vector search / RAG 벡터 검색 |
| `features.mcp` | MCP server/client (JSON-RPC 2.0) / MCP 서버/클라이언트 |
| `features.mcp_marketplace` | MCP marketplace (install/catalog) / MCP 마켓플레이스 |
| `features.a2a` | Agent-to-agent protocol / 에이전트 간 프로토콜 |
| `features.agents` | Multi-agent management / 멀티에이전트 관리 |
| `features.self_evolve` | Self-evolution (prompt/tool improvement) / 자기 진화 |
| `features.thoughts` | Thought tracking / 사고 추적 |
| `features.mood` | Mood tracking & expression / 감정 추적 및 표현 |
| `features.model_detect` | Auto model detection / 자동 모델 감지 |
