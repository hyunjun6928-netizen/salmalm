# Features Overview
# 기능 개요

SalmAlm includes 10 major feature categories that make it a complete personal AI gateway.

SalmAlm은 완전한 개인 AI 게이트웨이를 만드는 10개의 주요 기능 카테고리를 포함합니다.

| # | Feature / 기능 | Description / 설명 |
|---|---|---|
| 1 | [Multi-Model Routing](multi-model.md) | Auto-select Opus/Sonnet/Haiku / 멀티모델 자동 라우팅 |
| 2 | [Extended Thinking](thinking.md) | Deep reasoning mode / 확장 사고 모드 |
| 3 | [RAG & Knowledge Base](rag.md) | Vector search + knowledge store / 벡터 검색 + 지식 저장소 |
| 4 | [MCP Integration](mcp.md) | Model Context Protocol / 모델 컨텍스트 프로토콜 |
| 5 | [Session Management](sessions.md) | Branch, rollback, groups / 분기, 롤백, 그룹 |
| 6 | [Security & Vault](security.md) | AES-256, OWASP, sandbox / 보안 |
| 7 | [Personal Assistant](personal.md) | Briefing, reminders, expenses / 브리핑, 리마인더, 가계부 |
| 8 | [Telegram & Discord](channels.md) | Multi-channel integration / 멀티채널 통합 |
| 9 | [Self-Evolution](self-evolve.md) | Auto-improvement / 자기 진화 |
| 10 | [SLA & Monitoring](sla.md) | Uptime, response time tracking / SLA 모니터링 |
